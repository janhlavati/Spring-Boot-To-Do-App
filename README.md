Spring Boot To-Do Application

Simple To-Do List web application built with Java Spring boot, designed for quick daily task management.

Features
The application itself provides basic interface for effective daily task organization. Some of the functions and features of To-Do App are:
- Create New Tasks
- Edit Tasks (its description or status)
- Delete Tasks
- Store tasks (tasks are stored in the database file)

Technologies
This application is built on a Spring Boot foundation usig Maven for dependency management. Core technologies:
- Language: Java 21
- Framework: Spring Boot 3.4.12
- Build tool: Apache Maven
- Database: H2
